# ToDoUp Privacy Policy

Last updated: March 6, 2026

ToDoUp stores account information, tasks, assistant history, and local app preferences so users can sign in and keep productivity data synchronized. Authentication is handled by Supabase Auth and application data is stored in Supabase Postgres.

The app uses this data to authenticate users, display dashboards, sync tasks, and maintain assistant-generated task drafts. ToDoUp does not sell user data. Access is restricted through authenticated requests and row-level security policies.

Users can export their data from the Settings screen and can request permanent deletion through the in-app delete-account flow. Support and privacy requests should be routed to the production support contact before public launch.
