## Repository layout

- `Enhance App Smoothness/flutter_mobile/` - main Flutter application

## What this repo delivers

- Task and productivity app workflows
- Supabase-based auth and task sync
- Reminders and daily engagement features
- Release-ready Flutter project structure

## Quick start

```powershell
cd "Enhance App Smoothness/flutter_mobile"
flutter pub get
flutter run
```
